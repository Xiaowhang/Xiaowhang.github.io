A. 计算分段函数

题目描述

本题目要求计算下列分段函数的值： \[ f(x)=\begin{cases} \sqrt{\vert x-5 \vert} & (x\lt 0) \\ 0 & (x=0) \\ \frac{7x+2}{(x+1)^7} & (0 \lt x \le 2) \\ \ln(x+5) & (x \gt 2) \\ \end{cases} \]

输入

输入在一行中给出实数 \(x\) 。

输出

在一行中按 \(f(x) = result\) 的格式输出，其中 \(x\) 与 \(result\) 都保留三位小数。

B. 成绩评级

题目描述

对输入的多个成绩进行评级，标准如下：

0~59分 判为 F；

60~69分 判为D；

70~79分 判为C；

80~89分 判为B；

90~100分 判为A；

小于0分或者大于100分，则提示为“Invalid”，并结束程序。

要求使用switch语句，不用该题不得分！！！

输入

在一行中输入多个分数，期间以空格分割。

输出

分别输出每个分数及对应评级，期间以“-->”相连

C. 字符合并

题目描述

给定一串字符序列（以 # 结束），按照单词将字符合并（字符序列中，每个单词的首字母已大写，非首字符均为小写字母）。合并后每个大写字母前有一个空格，第一个字符前没有空格，最后一个字符后没有空格。请输出包含不超过 \(N\) 个合并后字符的序列

注意：如果 \(N\) 大于合并后(包括两个单词之间保留的空格)的字符个数，则输出完整的字符序列，否则只输出 \(N\) 个合并后的字符就停止

输入

输出一个正整数 \(N\)

输入字符数据，并保证在 # 之前不会出现换行符 \(\textbackslash n\)

输出

最多输出 \(N\) 个合并后的字符序列

D. 统计奇数字相邻的自然数的个数

题目描述

输入一个自然数（非负整数 \(n\) )，统计从 \(0\) 到这个数(包括这个数 \(n\) ) 范围内出现“奇数字相邻的自然数”的个数，若没有则输出 \(None\)

“奇数字相邻的自然数”指的是将这个自然数逐位分割后，有两个或以上相邻的数字是奇数。例如：\(10150\) 是奇数字相邻的自然数，\(123\) 则不是。

若输入 \(20\) ，在 \(0\) 到 \(20\) 的范围内，奇数字相邻的自然数有 \(11,13,15,17,19\) ，共 \(5\) 个。

输入

整型范围内的非负整数 \(n\)

输出

从 \(0\) 到这个数(包括这个数 \(n\) ) 范围内出现“奇数字相邻的自然数”的个数，若没有则输出 \(None\)

Z. 统计奇数字相邻的自然数的个数

题目描述

技术大学即将迎来建校五周年校庆，为了祝贺学校建立五周年，校方决定新建一栋教学楼（我也不知道能建在哪）。

施工队现在有 \(a\) 个 \(A\) 材料和 \(b\) 个 \(B\) 材料，而建第 \(i\) 层楼需要 \(i\) 个 \(A\) 材料和 \(i\) 个 \(B\) 材料。

但是校方觉得这样的教学楼还不够高，于是又拿出了 \(c\) 块钱，每块钱都可以用来买 \(1\) 个 \(A\) 材料或者 \(1\) 个 \(B\) 材料。

在校学生杰姆想知道，最后的教学楼最多能建多少层。

输入

第一行输入三个整数 \(a,b,c\) ，以空格隔开。 \((0 \le a,b,c \le 1e6)\)

输出

一行且只有一个整数，表示教学楼最多能建多少层。
