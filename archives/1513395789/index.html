A

思路

只有两个相同的字符在一起，才算特殊字符，而三个相同的在一起，中间的不算特殊字符

故特殊字符是两两同时出现的。

当 \(n\) 为奇数时不存在。

当 \(n\) 为偶数时每次出现两个相同的且与上一次不同的字符即可

Code

B

思路

从后往前判断，因为将一个数拆分只会变小，如果已经比后面的小，拆分的操作不会对后面有影响

如果需要拆分需判断拆分后是否合理

一直记录当前的大小即可

移动两次后再进队
