训练模型的样本数据集检查

题目描述

在人工智能领域，通常需要对用于训练模型的样本数据集进行检查，如果有明显偏离样本的整体值的数据点（称为异常值，也称离群点，outlier)，就需要去掉或找一个新的数据替代这个离群点，以免影响训练模型的精度。通常可以用下述方法来检测样本的离群值。

对组成样本的数据 \(X = \{x_0, x_1, \dots , x_{N-1}\}\) （总计 \(N\) 个数， \(3 \le N\) ）进行抽样（抽取x中一部分数据组成新的数据集，例如把怀疑是离群点的数据去掉)。对新抽样样本 \(Y = \{ y_0, y_1, \dots ,y_{M-1} \}\)（总计 \(M\) 个数， \(2 \le M \le N－1\)），计算其平均值， \[ \mu = \frac{1}{M}\sum^{M - 1}_{i = 0}y_i \] 和 \(Y\) 的标准差， \[ \sigma = \sqrt{\frac{\sum^{M - 1}_{i = 0}(y_i - \mu )^2}{M - 1}} \] 对于原样本 \(X\) 中的数据 \(x_i\) （其中 \(i\) 为其在 \(X\) 的位置），如果满足 \(x_i \lt \mu - 3\sigma\) 或者 \(x_i \gt \mu + 3\sigma\) ，则可以认为 \(x_i\) 是原样本数据集 \(X\) 的离群点。对于离群点需要找一个数替代它，使得样本 \(X\) 的标准差最小。

从键盘上输入 \(n\ (3 \le n \le 1000)\) 个实数组成一个数据集，使用上述方法确认其中是否含有离群点，如果没有输出 None.；反之则输出找到的第一个离群值（保留两位有效数字）、它在这个数据集中的位置和替代它的实数（保留两位有效数字）。

例如：

  1. 输入的 \(7\) 个数是：

    0.012513, 5.635853, 1.933042, 8.087405, 5.850093, 4.798730, 3.502915。

    则输出：

    \(None.\)

  2. 输入的 \(13\) 个数是：

    0.006256, 2.817927, 0.966521, 4.043703, 2.925047, 2.399365, 1.751457, 4.479812, 4.114200, 3.733024, 25.000000, 4.294717, 3.552507。

    则输出：

    \(25.00,\ 10,\ 2.92\)
